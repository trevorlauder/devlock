## What devlock is

devlock runs a coding agent inside a Linux sandbox built for local development. The agent sees only the project you point it at, cannot reach the network except through a proxy you control, and never gets your real API credentials. You configure it with two YAML files. One describes the agent, one describes what the sandbox lets through.
